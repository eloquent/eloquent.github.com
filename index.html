Projects

Eloquent Software provides free, high-quality, open-source software components and applications for public use. These projects are the result of many crazy ideas, sleepless nights, intense discussions, and hard working people. We hope you find them useful.

Asplode Drop-in exception-based error handling for PHP.

Asplode is very simple library that sets up an error handler to throw ErrorException exceptions instead of using the default PHP error handler.

Dumpling Diff-friendly mysqldump with an improved interface.

Dumpling is a wrapper for mysqldump that uses sensible defaults, and provides a streamlined interface. Dumpling is designed to be as intuitive and simple to use as possible.

Composer configuration reader A light-weight component for reading Composer configuration files.

Composer configuration reader provides a light-weight component for reading Composer configuration files. It is intended for use when including the entire Composer package as a dependency is overkill for the needs of a system.

Composer NPM bridge NPM integration for Composer packages.

The Composer NPM bridge allows installation and updating of NPM modules via the Composer command line interface. This allows Composer packages to bring in NPM dependencies in a similar fashion to regular Composer dependencies. The bridge uses NPM's shrinkwrap system to achieve similar functionality to Composer's lock files.

Cosmos A library for representing and manipulating PHP class names.

Cosmos is a library for representing and manipulating PHP class names. It includes a ClassName object, as well as tools to help in resolving class names against a set of namespace and use statements.

Enumeration An enumeration implementation for PHP.

In terms of software development, an enumeration (or "enumerated type") is essentially a fixed set of values. These values are called "members" or "elements". An enumeration is used in circumstances where it is desirable to allow an argument to be only one of a particular set of values, and where anything else is considered invalid.

Equality A better strict comparison for PHP.

Equality is a library for comparing PHP values in a deep and strict manner, without requiring object values to be the exact same instance.

Fixie YAML-based data fixtures.

Fixie is a format for storing tabular data. It blends the strengths of YAML and CSV to produce a syntax that is well suited for both human and machine readers. The Fixie syntax is actually a subset of YAML 1.2, meaning that any given example of Fixie syntax is also perfectly valid YAML. Unlike free-form YAML however, all Fixie variants can be read row-by-row, which allows for minimal memory usage when reading large amounts of data.

Liberator A proxy for circumventing PHP access modifier restrictions.

Liberator is an object proxy that allows you to access methods and properties of an object or class that would normally be restricted by PHP. Essentially, this means you ignore any 'private' or 'protected' keywords. Liberator's primary use is as a testing tool. Unit tests can often be simplified using a mix of partially mocked objects and Liberator. Liberator is based upon the Pops object proxy system.

Liftoff Open any file or URI using the default GUI application from within PHP.

Liftoff provides a simple interface to launch a file or URI in the host operating system's default application, in a cross-platform manner. Liftoff can be used, for example, to launch a HTML page in the operating system's default browser from within a command line PHP application.

Lockbox Simple, strong encryption.

Lockbox is the simplest possible way to implement strong, two-way, public-key encryption for use in applications. Lockbox uses a combination of well-established technologies to ensure the safety of data. Lockbox libraries exist for multiple languages; see the Lockbox website for more information.

Mooch A simple Twitter OAuth proxy.

Mooch is a simple app designed to allow unauthenticated access to the Twitter API for web apps that have no server-side components. Mooch is designed to be deployed as a Heroku app, which makes deployment of a new Mooch service extremely simple.

Otis One-time password / two-factor authentication library for PHP.

Otis is a PHP library for implementing one-time password / two-factor authentication systems. Otis provides generators and validators for both TOTP (time-based passwords as defined in RFC 6238) and HOTP (counter-based passwords as covered in RFC 4226). Otis supports all hashing algorithms (SHA-1, SHA-256, SHA-512).

Pathogen General-purpose path library for PHP.

Pathogen is a library for path manipulation. Pathogen supports file system paths including Unix and Windows style paths, but is truly a general-purpose path implementation, capable of representing URI paths and other path-like structures while providing a comprehensive API.

PBKDF2-compat A compatibility library for the PHP 5.5 function hash_pbkdf2().

This library provides a forwards-compatible implementation of the hash_pbkdf2() function introduced in PHP 5.5. It allows the use of this function in versions of PHP prior to 5.5, and falls back to the native function when it is available.

Pops PHP object proxy system.

Pops is a system for wrapping PHP objects in other objects to modify their behaviour. A Pops proxy will, as much as possible, imitate the object it wraps. It passes along method calls and returns the underlying result, and allows transparent access to properties (for both setting and getting). Pops is the underlying system behind Liberator.

Typhax A flexible PHP type hinting syntax.

Typhax is a standard for the specification of PHP types in parameter type hints, and anywhere a type needs to be described in a human-readable form. It expands on the capabilities of existing conventions for specifying type requirements, such as those used in the PHP documentation and PHPDoc's @param tag. In addition to scalar and class type hints, Typhax allows powerful features, including the specification of key and value types for arrays, as well as compound types that use boolean logic (e.g. integer|float).
